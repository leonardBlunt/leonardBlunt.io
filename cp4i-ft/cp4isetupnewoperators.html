Install Operators for Integration Capabilities

  1. Overview
  2. Installation of new operators

Note:This Class Material is designed for Cloud Pak for Integration 2023.2 use the following operator versions and set automatic update to false. Operator Versions



We will need to install the operators for the integration capabilties. These operators to be installed include:

  1. IBM Messaging Queue
  2. IBM Event Streams
  3. IBM Event End Point Mangement
  4. IBM API connect
  5. IBM App connect
  6. IBM Foundation Assets

  1. IBM Mesaging Queue

IBM MQ is messaging middleware that simplifies and accelerates the integration of diverse applications and business data across multiple platforms. It uses message queues to facilitate the exchanges of information and offers a single messaging solution for cloud, mobile, Internet of Things (IoT) and on-premises environments.

The IBM MQ Operator for Red Hat OpenShift Container Platform provides an easy way to manage the life cycle of IBM MQ queue managers.

  1. Go to OCP Console -> Operators-> Operator Hub -> Choose the project cp4i -> filter by IBM MQ
  2. Click Install
  3. We will need to choose the various options for the installation. You may refer to the details below. Note: It would take couple of minutes for the installation to complete.
  4. You will see a message of successful completion when the installation is complete.
  5. You can go to oc oc CLI to check on the installed pods.
    6. oc get pods
