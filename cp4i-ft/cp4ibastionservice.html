Bastion Service (Optional)

Creating a bastion service in Red Hat OpenShift involves setting up a secure entry point to access and manage resources within your cluster. This service is intended as a contingency plan to be utilized only in the event that the practicum bastion server is inaccessible during the practicum session.

Creating Bastion Service

  1. Go to Openshift Console -> Click "+"
  2. Choose -> "All Projects". Add the following code snippet and click "Create"
  3. Go to Workloads -> Pods -> Project -> "All Projects" -> search for "Bastion-service". Note: Wait till you see the pod running state/status.
  4. Click the pod related to bastion-service -> Terminal
    Note:This bastion service is in raw form and hence you will need to install basic libraries such as "wget".
    5. yum install -y wget
