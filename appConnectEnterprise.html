Configure the App Connect Enterise and Designer

  1. Overview

In this section we will configure the App Connect Enteripise and Integration Design. We will use the PlatformINtegration UI to quickly configure a Integration Dashboard for running Integration Servers and a Integration Design instance that will support the Low code developent of the API to MQ using the integration connectors.


  1. Configure Integration Dashboard

In this section we will configure the integration dashboard.


  1. Navigate to the Integration Instances page from the Platform UI home page


  2. Click on the Create Instance button. On the Create Instance page you will see the integration capabilities that can be configured. Greyed out capabilities is an indicator that the operators for these capabilities are not yet installed.

  3. Select Integration Dashboard and Click Next


  4. Select the Quickstart template and click Next


  5. For configuring this quick start instance we will name the instance "ace-db" and accept the license then leave otehr configuration options unchanged.


  6. Wait for the integration Dashboard to change status to Ready.
